Материалы

В Инспекторе, в компоненте Рендер Сетки, в разделе Материалы перечислены все материалы, которые использует рендер сетки. Сетки, импортированные из программ 3D-моделирования, могут использовать несколько Материалов и каждая подсетка использует один Материал из списка.

Если сетка содержит больше материалов, чем подсеток, Unity рендерит последнюю подсетку с каждым из оставшихся материалов, один поверх другого. Это позволяет настроить многопроходный рендеринг на этой подсетке. Однако это может повлиять на производительность во время выполнения. Полностью непрозрачные Материалы перезаписывают предыдущие слои, что приводит к снижению производительности без каких-либо преимуществ.

Свойства: Функции:
Размер Укажите количество Материалов в Рендере Сетки. Если уменьшить размер списка Материалов, Unity удаляет элементы с конца списка.
Элемент Список Материалов в Рендере Сетки в числовом порядке. Первый элемент всегда называется Element 0.

Освещение

Раздел Освещение содержит свойства того как этот Рендер Сетки взаимодействует с освещением в Unity.

Свойства: Функции:
Отбрасывание теней (Cast Shadows) Укажите, отбрасывание тени для сетки, когда на нее светит подходящий Свет.
On Сетка отбрасывает тень, когда на нее светит Свет, создающий тень.
Off Сетка не отбрасывает тени.
Двухсторонний (Two Sided) Сетка отбрасывает тени с обеих сторон. Чтобы Enlighten Realtime и Baked Illumination и Прогрессивная Светокарта поддерживали двусторонние тени, материал, назначенный целевому рендереру сетки, должен поддерживать Double Sided Global Illumination.
Только тени Тени от сетки видны, но не сама сетка.
Receive Shadows Включите эту опцию чтобы сетка отображала все отбрасываемые на нее тени. Поддерживается только при использовании Прогрессивного Лайтмапа.
Глобальное освещение Выберите это свойство, чтобы включить флаг Глобального Освещения в Static Editor Flags игрового объекта. Это свойство включает рендерер сетки данного игрового объекта в расчеты глобального освещения. Эти вычисления происходят при предварительном расчете данных об освещении во время запекания. Это свойство действует только в том случае, если вы включили для этой сцены такие настройки глобального освещения, как Baked Global Illumination или Enlighten Realtime Global Illumination. Это свойство включает опции получения глобального освещения.
