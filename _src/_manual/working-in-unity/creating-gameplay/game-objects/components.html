--- title: Введение в компоненты permalink: /Manual/Components.html ---

Игровой объект - это объект в редакторе Unity, который содержит компоненты. Компоненты определяют поведение этого игрового объекта.

На этой странице описано как просматривать и взаимодействовать с компонентами, а также представлен краткий обзор наиболее распространенных конфигураций компонентов в Unity.

Чтобы просмотреть компоненты игрового объекта, выберите игровой объект в Scene View или Hierarchy, затем откройте окно Inspector для просмотра списка всех компонентов и их настроек.

Вы можете взаимодействовать с компонентами непосредственно в редакторе или с помощью скрипта. Руководство по управлению и взаимодействию с компонентами с помощью скриптов см. в разделе "Скрипты".

Общие конфигурации компонентов

В этом разделе подробно описаны некоторые фундаментальные конфигурации компонентов по умолчанию в Unity.

Компонент Transform

Каждый Игровой Объект в Unity имеет компонент Transform. Этот компонент определяет положение, вращение и масштаб игрового объекта в Игровом Мире и Scene View. Вы не можете удалить этот компонент.

Компонент Transform также позволяет реализовать концепцию родительских отношений, которая позволяет вам сделать игровой объект дочерним объектом другого игрового объекта и управлять его положением с помощью компонента Transform родительского объекта. Это фундаментальная часть работы с игровыми объектами в Unity.

Компонент Main Camera

По умолчанию каждая новая сцена начинается с игрового объекта под названием Main Camera. Этот игровой объект настроен для работы в качестве основной камеры в вашей игре. Он содержит компонент Transform, компонент Camera и компонент Audio Listener для приема звука в вашем приложении.
