--- title: Сцена permalink: /Manual/CreatingScenes.html ---

Сцены - это место, где вы работаете с контентом в Unity. Это активы, которые содержат всю или часть игры или приложения. Например, простую игру можно создать в одной сцене, а для более сложной игры можно использовать по одной сцене на уровень, каждая из которых имеет свое окружение, персонажей, препятствия, декорации и пользовательский интерфейс. В проекте можно создать любое количество сцен.

Когда вы создаете новый проект и открываете его в первый раз, Unity открывает образец сцены, содержащий только Camera и Освещение.

Информацию о работе со сценами см. в разделе Создание, загрузка и сохранение сцен.

Шаблоны сцен

Unity использует шаблоны сцен для создания новых сцен. Шаблоны сцен - это активы, которые хранятся в проекте. Они похожи на сцены, но предназначены для копирования, а не для прямого использования.

Информацию о создании и использовании шаблонов сцен см. в разделе Шаблоны Сцен.

Окно New Scene

Окно New Scene открывается при создании новой сцены из меню File: File -> New Scene или комбинацией клавиш Ctrl/Cmd + N. Используйте его для создания новых сцен на основе определенных шаблонов сцен в вашем проекте, а также для получения информации о существующих шаблонах.

  1. Поле поиска: найти доступные шаблоны сцен по названию.
  2. Шаблоны: список всех доступных шаблонов в проекте.
  3. Информация о шаблоне: отображение информации о текущем выбранном шаблоне.
  4. Командная строка: содержит команды и опции для создания новой сцены на основе выбранного шаблона.

Создание новой сцены

Чтобы создать новую сцену в окне New Scene, выберите шаблон из списка шаблонов и нажмите кнопку Create. Подробное описание создания сцены см. в разделе Создание новой сцены в окне "New Scene".

Прикрепленные шаблоны

Прикрепленные шаблоны появляются перед другими шаблонами в списке шаблонов окна "New Scene". Последний прикрепленный шаблон отображается в верхней части списка.

Чтобы закрепить шаблон, щелкните по иконке с "булавкой".

Вы также можете закрепить шаблон при редактировании его свойств. Включите опцию Pin в окне New Scene.

Расположение и редактирование шаблонов

При выборе шаблона в окне "New Scene" на панели информации отображается путь к шаблону (1 на рисунке ниже).

Чтобы выделить шаблон в Project Window, выберите ссылку Locate (2 на рисунке ниже).

Чтобы открыть шаблон в окне Inspector и отредактировать его свойства, выберите ссылку Edit (3 на рисунке ниже).

Редактирование множества сцен

Вы можете одновременно открыть несколько сцен для редактирования. Подробнее см. в разделе редактирование нескольких сцен.
