На этой странице описано как создавать, загружать и сохранять сцены.

Создание новой сцены

Существует несколько способов создания новой сцены:

  • Используйте окно New Scene для создания новой сцены на основе определенного шаблона сцены.
  • Используйте Menu или Project Window для создания новых сцен на основе шаблона базовой сцены проекта без открытия окна New Scene.
  • Создайте сцену при помощи скрипта C#.

Unity создает каждую новую сцену на основе шаблона сцены. Информацию о создании и управлении шаблонами сцен см. в разделе Шаблоны сцен.

Создание новой сцены с помощью окна New Scene

Используйте окно New Scene для создания новых сцен на основе определенных шаблонов сцен в вашем проекте. Вы также можете использовать окно New Scene для поиска и управления шаблонами сцен. Подробнее см. в разделе Окно New Scene.

По умолчанию окно New Scene открывается при создании новой сцены из Menu (File > New Scene) или с помощью сочетания клавиш (Ctrl/Cmd + N).

Чтобы создать новую Сцену:

  1. Выберите шаблон из списка.
  2. Если вы хотите, чтобы Unity дополнительно загружала новую сцену (см. примечание ниже), включите опцию Load Additively.
  3. Нажмите кнопку Create, чтобы создать новую сцену на основе шаблона.

Примечание: дополнительная загрузка означает, что Unity загружает сцену в дополнение к любым другим открытым сценам. Дополнительные сведения см. в разделе редактирование нескольких сцен.

Если у шаблона нет клонируемых зависимостей, Unity загружает новую сцену в память, но не сохраняет ее.

Если шаблон имеет клонируемые зависимости, Unity предложит вам выбрать место для сохранения. Когда вы сохраняете сцену, Unity создает папку в том же месте и с тем же именем что и новая сцена. Затем она клонирует клонируемые зависимости в новую папку и обновляет новую сцену, чтобы использовать клонированные активы вместо оригинальных активов, использованных в сцене-шаблоне.

Создание новой сцены из Menu:

Используйте Menu Assets -> Create -> Scene для создания новой сцены без открытия окна New Scene.

Когда вы создаете новую сцену из Menu, Unity автоматически копирует основной шаблон проекта и добавляет новую сцену в ту папку, которая в данный момент открыта в Project Window.

Создание новой сцены из Project Window

Используйте контекстное Menu в Project Window, чтобы создать новую сцену без открытия окна New Scene.

  1. Перейдите в папку, в которой вы хотите создать новую сцену.
  2. Щелкните правой кнопкой мыши по папке в левой панели или щелкните правой кнопкой мыши пустую область в правой панели и выберите в контекстном Menu Create > Scene.

Когда вы создаете новую сцену из Menu, Unity автоматически копирует основной шаблон проекта и добавляет новую сцену в выбранную папку.

Создание новой сцены из C# скрипта

Чтобы создать новую сцену из скрипта C# с использованием определенного шаблона сцены, используйте метод Instantiate. 

Tuple<Scene, SceneAsset> SceneTemplate.Instantiate(SceneTemplateAsset sceneTemplate, bool loadAdditively, string newSceneOutputPath = null);

Метод Instantiate создает новую сцену из шаблона сцены. Он возвращает обработчик только что созданной сцены и соответствующий ей набор SceneAsset. Вы можете создать эту сцену как дополнительную. Если сцена содержит активы, которые должны быть клонированы, вы должны указать путь, по которому Unity сохранит сцену на диск.

События на новой сцене

Когда вы создаете новую сцену из шаблона, либо из скрипта, либо с помощью окна New Scene, Unity запускает событие. Unity запускает это событие после создания экземпляра шаблона, а также после запуска событий EditorSceneManager.newSceneCreated или EditorSceneManager.sceneOpened. 

Tuple<Scene, SceneAsset> SceneTemplate.Instantiate(SceneTemplateAsset sceneTemplate, bool loadAdditively, string newSceneOutputPath = null);

Загрузка сцены

Чтобы открыть сцену, выполните одно из следующих действий:

  • В Project Window дважды щелкните на активе сцены.
  • В Menu выберите File > New Scene
  • В Menu выберите File > Recent Scenes > [НАЗВАНИЕ-ФАМИЛИЯ].

Если текущая сцена содержит несохраненные изменения, Unity предложит вам сохранить сцену или отменить изменения.

Редактирование нескольких сцен

Вы можете одновременно открыть для редактирования несколько сцен. Подробнее см. в разделе редактирование нескольких сцен.

Сохранение сцен

Чтобы сохранить сцену, над которой вы сейчас работаете, выберите в Menu File > Save Scene или нажмите Ctrl + S (Windows).
